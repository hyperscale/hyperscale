The Hyperscale Programming Language
===================================

This is the main source code repository for Hyperscale. It contains the compiler, standard library, and documentation.

Install
-------

~~~
mkdir build
cd build/
cmake ../
make
make install
~~~

How-To
------

Build project
~~~
cd my-project/
hyperscale build
~~~

Test project
~~~
cd my-project/
hyperscale test
~~~
